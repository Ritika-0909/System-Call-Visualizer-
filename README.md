# 💻 System Call Visualizer

A full-stack web application that provides an interactive visualization of operating system system calls. The application helps users understand how user processes interact with the operating system kernel by displaying system call execution, process flow, and system-level operations through an intuitive graphical interface.

---

## 📌 Features

- 📊 Interactive visualization of system calls
- ⚙️ Displays execution flow between user processes and the kernel
- 🖥️ User-friendly React interface
- 🔄 Flask backend for processing and API services
- 📈 Graphical representation of system interactions
- 🚀 Fast and responsive web application
- 📚 Educational tool for Operating Systems concepts

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/Ritika-0909/System-Call-Visualizer.git
```

### Navigate to the project directory

```bash
cd System-Call-Visualizer
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

The frontend will run on:

```
http://localhost:5173
```

---

## Backend Setup

```bash
cd backend
pip install -r requirements.txt
python app.py
```

The Flask backend will start on:

```
http://127.0.0.1:5000
```

---

## How It Works

1. Start the Flask backend.
2. Launch the React frontend.
3. The frontend sends requests to the Flask API.
4. The backend processes system call data.
5. Results are displayed through an interactive visualization.
